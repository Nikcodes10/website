# This is the official website of IETE-SF-MEC

## Steps to run locally

First, `cd` into the root directory after cloning repo.

Then run:

```bash
npm install
# and
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
